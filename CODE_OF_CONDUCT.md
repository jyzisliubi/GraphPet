# Contributor Covenant Code of Conduct

## 我们的承诺

为了营造一个开放友好的环境，所有贡献者和维护者承诺让 GraphPet 社区的每一位参与者免受骚扰，无论年龄、体型、可见或不可见的残障、族裔、性别认同与表达、性倾向、社会地位、教育程度、国籍、相貌、种族、宗教或性取向如何。

我们承诺以同理心、尊重和友善的态度行动和沟通。

## 我们的标准

正面行为示例：

- 对他人展示同理心与友善
- 尊重不同观点与经验
- 给予建设性反馈
- 接受责任并向他人道歉
- 关注的不仅是个人，更是整个社区

不可接受的行为：

- 使用色情化语言或 imagery、不受欢迎的性关注
-钓鱼、辱骂、贬损评论，人身或政治攻击
- 公开或私下骚扰
- 未经明确许可发布他人私人信息
- 其他违反职业道德的行为

## 执行

违规行为可邮件至 `3364415961@qq.com` 举报，所有举报都会被审慎公正地处理。
